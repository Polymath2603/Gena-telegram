# Gena Bot ü§ñ

Gena is a powerful Telegram bot powered by Google's Gemini AI, designed to be your intelligent AI companion.

## üöÄ Features

- **AI Conversations**: Chat with Gemini directly on Telegram.
- **Vision Support**: Send images and ask questions about them.
- **Custom Personas**: Switch between different AI personalities using `/settings`.
- **System Commands**: Use `/clear` to reset context and `/admin` for dashboard access (admins only).
- **Payment Integration**: Support for Telegram Stars subscription payments.

## üìã Quick Start

### Prerequisites
- Python 3.8+
- Telegram Bot Token (from @BotFather)
- Google Gemini API Key (from Google AI Studio)

### Installation
```bash
git clone https://github.com/yourusername/gena-bot.git
cd gena-bot
pip install -r requirements.txt
```

### Configuration
1. Copy `.env.example` to `.env`.
2. Fill in your `TELEGRAM_TOKEN`, `GEMINI_API_KEY`, and `ADMIN_USER_IDS`.

### Usage
```bash
python telebot.py
```

## üõ†Ô∏è Components
- `telebot.py`: Main entry point for the Telegram bot.
- `gena.py`: Core logic for Gemini API interaction.
- `database.py`: User data and session management.
- `admin_dashboard.py`: Admin statistics and control panel.
